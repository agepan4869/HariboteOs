●これはなに？
　
　泊何水氏(http://yu_izumi.at.infoseek.co.jp/)が製作している
　「出水ゴシック体-M 16ドットフォント(IZMG16-N)」をOSASKに移植し、自分の
　趣味にあった自作半角フォント(NEC PC-9801シリーズの半角文字配列準拠)を
　くっつけたものです。
　JPN16V00.FNTで表示できる全ての文字(JIS X 0213:2004 1面 + 制御コードと
　特殊記号を含む半角文字)を表示できます。
　
　半角部分を泊何水氏の「Izumi Univers 16 dot」にしたファイルもセットで
　同封してあります。お好きな方をお使いください。
　
●中に入ってるもの
　
　JPN16V00_K.FNT = フォント本体です(半角部分が自作のもの)。stk5圧縮です。
　JPN16V00_K.CHR = ↑の圧縮前のファイルです。
　JPN16V00_I.FNT = フォント本体です(半角部分がIzumi Universのもの)。stk5圧縮です。
　JPN16V00_I.CHR = ↑の圧縮前のファイルです。
　HANKAKU.CHR ==== 自作半角部分だけのファイル。
　AFTER.PNG ====== 本フォントのサンプルです(!GUIDE!.TXT)。上=自作半角、下=Izumi Univers。
　README.TXT ===== これ。
　IZG16B60.DOC === 泊何水氏のドキュメントです。
　LICENSE.TXT ==== 川合堂ライセンス-01  ver.1.0
　
　
●使い方
　JPN16V00_K.FNTかJPN16V00_I.FNTの好きな方を「JPN16V00.FNT」にリネームして、
　OSASKのディスクにコピーするだけです。これで、日本語フォントを使う各ソフトから
　このフォントが使えるようになります。
　また、Kさん著の「３０日でできる！　ＯＳ自作入門」読者の方は、
　612ページ辺りを参考にすることで、最新の出水ゴシック16フォントに対応させる
　ことができます。是非挑戦してみて下さい。
